CPU - Central Processing Unit

The CPU is the brain of the computer. It performs calculations and executes instructions of a computer program.

Key features:

  • Speed measured in GHz
  • Number of cores (dual-core, quad-core, etc.)
  • Cache size

Installation:

  1. Open the CPU socket on the motherboard
  2. Align the CPU correctly and gently place it into the socket
  3. Close the CPU socket latch to secure the CPU in place
Example: Intel Core i7-9700K, 3.6GHz, 8 cores
GPU - Graphics Processing Unit

The GPU is responsible for rendering images and videos. It is crucial for gaming and graphical applications.

Key features:

  • GPU architecture (NVIDIA, AMD)
  • Video RAM (VRAM)
  • CUDA cores (NVIDIA GPUs)

Installation:

  1. Insert the GPU into the PCIe slot on the motherboard
  2. Secure the GPU to the case using screws
  3. Connect power cables from the PSU to the GPU
Example: NVIDIA GeForce RTX 3080, 10GB GDDR6X
RAM - Random Access Memory

RAM is a type of computer memory that is used to store working data and machine code currently in use.

Key features:

  • Capacity measured in gigabytes (GB) or terabytes (TB)
  • Speed measured in megahertz (MHz) or gigahertz (GHz)
  • Types: DDR3, DDR4, etc.

Installation:

  1. Locate the RAM slots on the motherboard
  2. Align the RAM module notches with the notches in the slot
  3. Gently push down on both ends of the RAM module until it clicks into place
Example: Corsair Vengeance LPX 16GB DDR4 3200MHz
Storage

Storage devices are used to store and retrieve digital information. Common types include HDDs and SSDs.

Key features:

  • Capacity measured in gigabytes (GB) or terabytes (TB)
  • Type: Hard Disk Drive (HDD) or Solid State Drive (SSD)
  • Connection interface: SATA, PCIe, etc.

Installation:

  1. Connect the storage device to the appropriate port on the motherboard
  2. Secure the storage device in the case using screws
  3. Connect power and data cables to the storage device
Example: Samsung 970 EVO Plus 1TB NVMe M.2 SSD
Motherboard

The motherboard is the main printed circuit board (PCB) in a computer. It houses the CPU, memory, and other essential components.

Key features:

  • Form factor: ATX, Micro-ATX, Mini-ITX, etc.
  • Socket type for CPU
  • Expansion slots: PCIe, RAM slots, etc.

Installation:

  1. Place the motherboard in the case
  2. Align the motherboard screw holes with the case standoffs
  3. Secure the motherboard to the case using screws
Example: ASUS ROG Strix B450-F Gaming ATX Motherboard
Power Supply (PSU)

The PSU provides electrical power to the computer components. It converts electrical power from an outlet into usable power for the computer.

Key features:

  • Wattage: measured in watts (W)
  • Modularity: modular, semi-modular, non-modular
  • Efficiency rating: 80 PLUS Bronze, Gold, Platinum, etc.

Installation:

  1. Place the PSU in the dedicated compartment in the case
  2. Secure the PSU to the case using screws
  3. Connect power cables to the motherboard, CPU, GPU, and storage devices
Example: EVGA SuperNOVA 750 G5, 80 Plus Gold 750W
Computer Case

The computer case houses and protects the internal components of the computer.

Key features:

  • Form factor compatibility
  • Number of drive bays
  • Cable management options

Installation:

  1. Open the computer case
  2. Place the components inside the case
  3. Connect front panel cables to the motherboard
Example: NZXT H510i Compact ATX Mid-Tower Case
Cooling Systems

Cooling systems prevent computer components from overheating. They include air and liquid cooling solutions.

Key features:

  • Fan size and speed
  • Radiator size (for liquid cooling)
  • Cooling block material (for liquid cooling)

Installation:

  1. Attach fans to the case for air cooling or mount a radiator for liquid cooling
  2. Connect fans or pump to the motherboard or PSU for power
  3. Apply thermal paste between CPU and cooler (for air or liquid cooling)
Example: Corsair Hydro Series H100i RGB Platinum SE Liquid CPU Cooler
